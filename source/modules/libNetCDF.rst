**Description for libNetCDF:**

Description from `NetCDF website: <http://www.unidata.ucar.edu/software/netcdf/>`_  "NetCDF is a set of software libraries and self-describing, machine-independent data formats that support the creation, access, and sharing of array-oriented scientific data."
