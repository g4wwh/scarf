**Using Python**

To use one of the latest versions of python run:

.. code-block:: console

  $ module load python/2.7.5
  $ python --version
  Python 2.7.5

There is also python 2.6.6 is also installed on the system. Without any modules loaded:

.. code-block:: console

  $ python --version
  Python 2.6.6
