**Description for libFFTW:**

Description from `FFTW website: <http://www.fftw.org/>`_  "FFTW is a C subroutine library for computing the discrete Fourier transform (DFT) in one or more dimensions, of arbitrary input size, and of both real and complex data (as well as of even/odd data, i.e. the discrete cosine/sine transforms or DCT/DST)."


**Using libFFTW:**

FFTW is available in a number of different versions corresponding to different combinations of fftw version, compiler version and whether or not mpi support is included. Choose the appropriate module to meet your requirements

Run:

.. code-block:: console

  $ module load libfftw/intel/3.3.3_mpi

to add FFTW libraries to your envrionment.
