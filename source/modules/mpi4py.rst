**Description for mpi4py:**

MPI binding for python.

**Using mpi4py:**

Please run launch an interactive session:

.. code-block:: console

  $ bsub -Is /bin/bash
  Job <1223> is submitted to queue <scarf>.
  <<Waiting for dispatch ...>>
  <<Starting on cnXXX.scarf.rl.ac.uk>>

And then run,

.. code-block:: console

  $ module load mpi4py/1.3
